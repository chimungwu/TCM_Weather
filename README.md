# 五運六氣臨床推算系統 (Yun Qi Calculator)

本系統是一款專為中醫臨床醫師與愛好者設計的專業工具，旨在將深奧的《黃帝內經》五運六氣理論轉化為直觀、精確的數位化推算結果。

## 📖 五運六氣理論簡介

「五運六氣」簡稱「運氣」，是中醫預測氣候變化規律及其對人體健康影響的核心理論。它結合了古代天文學、氣象學與醫學，認為宇宙間的氣候變化具有週期性，並直接影響人體的生理與病理。

### 1. 五運 (The Five Movements)
五運指木、火、土、金、水五行的運動。
*   **大運 (中運)**：主管全年的氣候總體趨勢。
*   **主運**：一年分五個時段（初、二、三、四、終運），每段約 73 日，代表常態氣候。
*   **客運**：隨年份變動，加臨於主運之上，代表氣候的變數。
*   **太過與不及**：陽年為太過（氣盛），陰年為不及（氣衰）。

### 2. 六氣 (The Six Qi)
六氣指風、寒、暑、濕、燥、火六種氣候因素。
*   **司天**：主管上半年（大寒至大暑）的氣候。
*   **在泉**：主管下半年（大暑至次年大寒）的氣候。
*   **主氣**：固定的季節性氣候（厥陰風木、少陰君火、少陽相火、太陰濕土、陽明燥金、太陽寒水）。
*   **客氣**：隨年份輪轉，加臨於主氣之上，是導致異常氣候（如暖冬、涼夏）的主要原因。

### 3. 運氣合德與特殊身分
當運與氣相遇時，會產生不同的化學反應：
*   **天符**：歲運與司天五行屬性相同。
*   **歲會**：歲運與歲支方位屬性相同。
*   **太乙天符**：天符與歲會同時出現，氣候變化最為劇烈。
*   **平氣**：當運氣相剋但具備特殊身分中和時，氣候趨於平和。

## 🛠 系統核心功能

*   **年度氣候總結 (SOP)**：自動生成歲次、中運、司天在泉的完整報告。
*   **主客加臨動態分析**：直觀呈現五運與六氣的「客主加臨」關係，並標註生剋順逆。
*   **《素問》治法與方藥**：根據《至真要大論》提供治則建議，並整合《三因極一病證方論》的運氣方劑。
*   **脈象診法參考**：根據司天在泉自動推算應見脈象，並提供「逆脈預警」。
*   **視覺化圓盤圖**：以傳統圓盤形式展現運氣層次，便於醫者觀察全局。
